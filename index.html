Auto Sub Sync by Pulasthi Mahawithana

AutoSubSync is an automatic subtitle synchronization tool. It recognizes the words spoken in the audio track using speech recognition libraries, and try to synchronize the existing erroneous subtitle track with the audio track.

Dependencies

  • Java
  • Sphinx4

Install

Currently works on linux platforms only. There is an issue in compiling of one of the libraries (CMU CAM toolkit) for windows. The windows version will be available as soon as the issue is fixed.
